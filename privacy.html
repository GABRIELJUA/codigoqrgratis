Política de Privacidad

La privacidad de los usuarios es una prioridad para esta herramienta. Esta página explica cómo se manejan los datos al utilizar el generador de códigos QR.

No recopilamos datos personales

Este sitio web no recopila, almacena ni procesa información personal de los usuarios. Toda la información introducida en el generador de códigos QR se utiliza únicamente para generar el código solicitado.

Los datos introducidos, como enlaces, redes WiFi, números de teléfono o textos, no se guardan en bases de datos ni se transmiten a servidores externos.

Procesamiento local

La generación de códigos QR se realiza completamente en el navegador del usuario mediante tecnologías web. Esto significa que el proceso ocurre localmente en tu dispositivo.

Debido a esto, ninguna información que ingreses en el generador es almacenada o analizada por este sitio web.

Uso de cookies

Este sitio puede utilizar almacenamiento local del navegador únicamente para guardar preferencias del usuario, como el modo oscuro. No se utilizan cookies de seguimiento ni sistemas de rastreo de usuarios.

Cambios en esta política

Esta política de privacidad puede actualizarse ocasionalmente para reflejar mejoras en el sitio o cambios en los servicios ofrecidos.

Se recomienda revisar esta página periódicamente para mantenerse informado sobre cualquier actualización.
